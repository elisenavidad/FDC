Storage Capacity and FDC Application Help Guide

This application calculates the storage capacity of a potential reservoir and flow-duration curve given a location at any stream within the Dominican Republic, a curve number and a dam height. Follow the steps below in order to successfully calculate storage capacity and a flow-duration curve.

  1. Zoom to location
    2. Zoom into a general region of interest.
  2. Select point of interest
    3. Click the "Click here to draw point" button to begin selecting a location. Next, click on a stream on the map to select your point of interest.
  3. Specify dam height
    4. Enter a desired dam height in the text box below "Enter Dam Height (Meters)". The default value is set to 100 meters.
  4. Specify Curve Number
    5. Enter a desired curve number in the text below "Curve Number". The default value is set to 80.
  5. Calculate storage capacity
    6. Click the "Submit" button to begin calculating storage capacity.
  6. Review results
    7. After clicking the "Submit" button, the application will produce a total volume in cubic meters, and draw both a watershed and a reservoir area on the map. The flow duration curve is found by clicking on the FDC Results button.
Note: Storage capacity will only be calculated if users click within a river stream. Clicking outside a stream will prompt a message asking users to specify a point of interest at least 100 meters within the closest stream.
